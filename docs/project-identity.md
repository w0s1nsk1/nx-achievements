# Project identity

## Name
**NXAchievements**

## One-line description
Offline-first achievement hub for Nintendo Switch homebrew users.

## Longer description
NXAchievements to lokalny framework achievementów działający jako homebrew app. Ładuje community packi JSON, śledzi odblokowania w lokalnym profilu i pokazuje postęp bez integracji z usługami Nintendo.

## Non-goals
- Brak integracji z Nintendo Switch Online
- Brak omijania auth Nintendo lub usług Nintendo
- Brak memory scanning/sysmodule/overlay w aktualnym zakresie
- Brak funkcji pirackich
- Brak online sync w obecnym milestone

## Safe wording (Switch homebrew / CFW)
- Używamy określeń: "homebrew app", "local viewer", "community packs".
- Nie sugerujemy, że app jest oficjalnym komponentem Nintendo.
- Nie używamy nazw/brandingu Nintendo Online do opisu funkcji projektu.

## Branding rules
- Oryginalne nazewnictwo i assety
- Zero logotypów Nintendo / NSO
- Styl może być inspirowany konsolowym UX, ale musi być autorski

## Terminology
- **achievement** — pojedyncze osiągnięcie z warunkami odblokowania
- **badge** — tier wizualny osiągnięcia (leaf/compass/bonfire/star)
- **pack** — plik JSON z definicją osiągnięć dla gry
- **profile** — lokalny plik z odblokowanymi achievementami
- **unlock** — zapis odblokowania achievementu z timestampem
