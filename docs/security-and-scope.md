# Security and scope

## Local-first
Data is stored locally (packs and profiles).

## No Nintendo service interaction
No NSO/eShop/auth integration.

## No telemetry
No built-in outbound telemetry.

## No DRM bypass
No DRM/security bypass mechanisms.

## No cheat engine functionality
NXAchievements is not a cheat framework.

## No piracy tooling
No piracy tooling or instructions.

## Optional future sync philosophy
If sync is added later, it should be opt-in, transparent, and independent from Nintendo services.
