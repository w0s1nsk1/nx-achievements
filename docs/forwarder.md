# Forwarder (HOME menu launcher) — safe path

NXAchievements działa jako homebrew `.nro`:
- `/switch/NXAchievements/NXAchievements.nro`

Użytkownik może **samodzielnie** utworzyć własny NSP forwarder, który tylko uruchamia powyższy plik z karty SD z poziomu HOME menu.

## Zasady bezpieczeństwa i brandingu
- Nie używaj nazwy „Nintendo Switch Online”.
- Nie używaj logotypów Nintendo ani NSO.
- Nie używaj żadnych cudzych/copyrighted assetów Nintendo.
- Forwarder ma wyglądać jak osobna aplikacja homebrew, nie jak usługa Nintendo.

## Rekomendowane metadane forwardera
- **Title name:** `NXAchievements`
- **Author/Publisher:** `NXAchievements Team`
- **Title ID:** custom/homebrew-safe placeholder (unikalny)

### Title ID guidance
- Używaj własnego, unikalnego ID i unikaj kolizji z innymi homebrew.
- Nie używaj ID należących do komercyjnych gier/aplikacji.

## Scope reminder
Ten dokument nie opisuje generowania forwardera krok po kroku i nie dostarcza gotowego NSP.
Projekt nie dostarcza:
- assetów Nintendo,
- instrukcji omijania zabezpieczeń,
- elementów związanych z piractwem.
