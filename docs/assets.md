# Asset guidelines

## Branding rules
- Ikony i grafiki muszą być **oryginalne**.
- Nie używać logotypów Nintendo.
- Nie używać ikonki/brandingu Nintendo Switch Online.
- Nie sugerować, że app jest oficjalną usługą Nintendo.

## Suggested visual direction
- Paleta może być inspirowana stylem red/white (Switch-like), ale projekt musi być autorski.
- Motyw achievementów:
  - `leaf`
  - `compass`
  - `bonfire`
  - `star`

## Planned assets
- `icon.png` (główna ikona app)
- ewentualny banner/store-preview (dla materiałów promocyjnych projektu)
- screenshoty viewer
